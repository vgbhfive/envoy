关于文档¶

Envoy 文档由以下几个主要的部分组成：

  • 介绍: 这部分主要包含了什么是 Envoy，架构预览，典型部署等内容。

  • 开始: 用 Docker 来快速开始使用 Envoy。

  • 安装: 如何用 Docker 来构建/安装 Envoy。

  • 版本历史: 每个版本的发布说明。

  • 配置: 关于 Envoy 详细配置的说明书。与此相关地，配置指导也包含了统计信息、运行时配置和 API 等。

  • 操作: 关于如何操作 Envoy 的常规信息，包括命令行、热重启包装器、管理界面和一个常规用的统计预览。

  • 扩展 Envoy: 关于如何书写 Envoy 自定义过滤器的信息。

  • API 参考: Envoy API 详细的参考信息。

  • Envoy FAQ: 有疑问？希望我们的答案能让你满意。
